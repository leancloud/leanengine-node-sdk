# LeanEngine Node.js SDK

## 安装

```bash
npm install leanengine --save
npm install leancloud-storage@3 --save
```

建议使用 Node.js 6.0 以上的版本（可在 `package.json` 中设置 `engines.node` 为 `6.x`）。
